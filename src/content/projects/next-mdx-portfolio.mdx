# My First Next.js Project 🚀  

This is a **portfolio project** built using **Next.js and MDX**. It allows dynamic content integration while keeping the structure simple and easy to manage.

## ✨ Features  

- **MDX Support**: Write content in MDX format with JSX.  
- **Syntax Highlighting**: Supports code blocks with styling.  
- **Custom Components**: Use React components inside MDX files.  
- **Responsive Design**: Works smoothly across all devices.  

## 🛠️ Technology Stack  

- **Next.js** – React framework for modern web apps.  
- **MDX** – Combines Markdown with JSX for flexibility.  
- **Tailwind CSS** – Utility-first styling for customization.  
- **Vercel** – Fast and efficient deployment platform.
