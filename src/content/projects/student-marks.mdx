# 🎓 Student Marks Management System

This project is a **full-stack web application** built using the **MERN stack**. It allows users to **enter and retrieve student marks** dynamically based on roll numbers. The system is designed to help beginners **understand backend development with Node.js** while ensuring a smooth **frontend experience with React**.

## 🚀 Features

- **Student Data Entry & Retrieval** – Users can store and fetch student marks.
- **RESTful API with Axios** – Handles API requests efficiently.
- **Dynamic Data Handling** – MongoDB ensures fast and reliable data storage.
- **CSS-Styled Forms & Pages** – Simple, clean, and responsive design.
- **Deployment on Render.com** – Hosted for easy access and testing.
- **Version Control with Git** – Code is managed and tracked efficiently.

## 🛠️ Technology Stack

- **MongoDB** – Stores student records efficiently.
- **Express.js** – Backend framework for handling API requests.
- **React.js** – Frontend framework for dynamic UI.
- **Node.js** – Server-side processing and API development.
- **Axios** – Fetches student data via HTTP requests.
- **CSS** – Provides a clean and structured UI.
- **Render.com** – Cloud platform for deployment.

## 📄 Pages  

1. **Home Page** – Landing page of the application.  
2. **Student Details Page** – Displays individual student marks.  
3. **Get Data Page** – Fetches student records dynamically.  

## 🏗️ Components  

The project includes **5 reusable components**, such as:  
- **Pop-ups** – Show notifications for user actions.  
- **Form loaders** – Improve user experience when submitting data.
